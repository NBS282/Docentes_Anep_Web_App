Bienvenido a la Plataforma de Planificación Escolar

Sigue estos sencillos pasos para comenzar a usar la aplicación y organizar tus planificaciones de forma rápida y sencilla.

Paso 1: Agregar Cursos

Para empezar, ve al menú y selecciona Crear para agregar los cursos en los que vas a realizar la planificación. Aquí puedes ingresar el nombre del curso, nivel y demás detalles importantes.

Paso 2: Crear Planificaciones

Una vez que hayas agregado los cursos, puedes comenzar a crear las planificaciones diarias o semanales. En el menú, selecciona Crear Planificación y elige el curso y el tipo de planificación que deseas crear.

Paso 3: Seleccionar Bloques de Contenido

Durante la creación de una planificación, selecciona los bloques de contenido y unidades de aprendizaje que se aplicarán en cada clase. Esto te ayudará a organizar las actividades y objetivos para cada sesión.

Paso 4: Imprimir o Compartir tu Planificación

Una vez que hayas terminado de crear la planificación, puedes optar por imprimir el documento, descargarlo en PDF , o incluso compartirlo con otros maestros para facilitar el trabajo en equipo.
